<h1>Chatbot Pirate Poétique</h1>

<h2>Description</h2>

<p>Ce projet a été développé dans le cadre d'un projet universitaire lors de ma deuxième année de bachelor en formation <strong>Data et Intelligence Artificielle</strong>. Il s'agit d'une application interactive de chatbot conçue pour répondre aux questions des utilisateurs sous forme de <strong>vers poétiques</strong> à la manière d'un pirate. Le projet utilise <strong>Python</strong> pour le développement, avec la bibliothèque <strong>Streamlit</strong> pour l'interface utilisateur et le modèle de langage <strong>Mistral</strong> pour générer des réponses créatives et poétiques. L'application permet aux utilisateurs de poser des questions et de recevoir des réponses en rimes, tout en affichant l'historique des échanges.</p>

<h2>Fonctionnalités</h2>

<ul>
    <li><strong>Réponses poétiques à thème pirate</strong> : Le chatbot est programmé pour toujours répondre sous forme de vers poétiques dans le style d'un pirate.</li>
    <li><strong>Interface utilisateur intuitive</strong> : Utilisation de <strong>Streamlit</strong> pour créer une interface web où les utilisateurs peuvent poser leurs questions et recevoir des réponses en temps réel.</li>
    <li><strong>Historique des conversations</strong> : Les échanges sont enregistrés dans l'état de session, permettant aux utilisateurs de revoir leurs interactions.</li>
</ul>
